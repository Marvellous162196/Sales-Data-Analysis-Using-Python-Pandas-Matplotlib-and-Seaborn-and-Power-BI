Sales Data Analysis Using Python and Power BI
Project Overview

This project analyzes a sales dataset using Python (Pandas, Matplotlib, and Seaborn) and Power BI. The objective is to identify sales trends, top-performing products, top-performing cities, and other business insights through data analysis and visualization.

Tools Used
Python
Pandas
Matplotlib
Seaborn
Power BI
Data Preparation

The dataset was cleaned and prepared by:

Removing unnecessary columns
Converting the Order Date column to datetime format
Creating new date-related columns such as Year, Month Name, Day, and Weekday
Checking for missing values and duplicates
Analysis Performed

The following business questions were answered:

Created a Month Name column showing the exact month names.
Identified the city with the highest sales.
Identified the product with the highest sales.
Separated the Order Date column into Day, Month, and Year.
Analyzed sales trends across different years.
Determined the day of the week with the highest sales.
Determined the month with the highest sales.
Explored additional insights through visualizations.

Key Findings

San Francisco recorded the highest sales among all cities.
MacBook Pro Laptop generated the highest sales revenue.
December was the highest-performing month.
Tuesday recorded the highest sales among all weekdays.
Most sales were recorded in 2019, while the 2020 data was limited.
Visualizations

Python visualizations were created using Matplotlib and Seaborn, while interactive dashboards were developed in Power BI to present key insights and trends.


Conclusion

The project demonstrates how Python and Power BI can be combined to clean, analyze, visualize, and communicate business insights from sales data. The findings can support better decision-making in sales strategy, marketing, and inventory planning.
